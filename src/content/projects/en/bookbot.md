## What it is

My **first Python project**, from the **Boot.dev** course. It reads a text file
(a book) and returns a report: how many words it has and how many times each
character appears.

## What I learned

- **Reading files** from Python and processing their content.
- Using **dictionaries** to count frequencies.
- Breaking a problem into **small functions** and understanding a program's flow
  from end to end.
- Running a Python program from the **terminal**.

## Why I include it

It's deliberately simple: it shows the fundamentals and was my starting point. It
gave me the base to take on bigger projects.
